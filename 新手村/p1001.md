----------------------
代码
```
#include <bits/stdc++.h>
using namespace std;
int main() 
{
    int a,b;
    cin >> a >> b;
    cout << a+b;
    return 0;
}
```
说明
```
int a,b;     //声明变量a,b  就是在内存中划定两个区域，叫a，b【注意 int 形式的声明变量只能存储整数（小数以后再说）】
             //解释：int 声明的变量之内存储整数！！！（such：  int a;  输入5.1  其实a里的值是5  自动舍去小数部分（没有四舍五入，小数部分统统丢掉））
```
